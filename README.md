# 🐍 PixelPython

**PixelPython** is a modern take on the classic Snake game, featuring a vibrant pixel-art design and smooth gameplay. Built with simplicity and performance in mind, this project blends nostalgia with a clean, modern user experience.

---

## Features

*  Classic Snake gameplay
*  Pixel-style UI and animations
*  Smooth controls and real-time movement
*  Dynamic food spawning system
*  Score tracking
*  Restart functionality after game over

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Tools:** VS Code, Git, GitHub

---


##  Installation & Setup

1. **Clone the repository**

```bash
git clone https://github.com/your-username/PixelPython.git
```

2. **Navigate to the project folder**

```bash
cd PixelPython
```

3. **Run the game**

* Open `index.html` in your browser
  **OR**
* Use Live Server in VS Code

---

##  How to Play

* Use **Arrow Keys** or **WASD** to move
* Eat the food  to grow longer
* Avoid:

  * Colliding with walls 
  * Colliding with yourself 
* Try to achieve the highest score possible
